# Описание MVP (Phase 1)

## Цель

Запуск безопасной версии бота для базового маркет-мейкинга:
- удержание стакана;
- сбор спреда;
- защита от MEV.

## Обязательный функционал

1. **Полноценный Trade Executor**
   - интеграция с Jito для отправки бандлов (Place + Cancel в одной транзакции);
   - атомарное размещение/снятие ордеров в рамках одного бандла.

2. **Запуск Pivot Engine**
   - подключение БД к реальному потоку сделок через WebSockets;
   - накопление истории сделок;
   - корректный расчет VWAP на основе накопленных данных.

3. **Kill Switch**
   - механизм экстренной остановки через Redis или файл-флаг;
   - при срабатывании — снять все ордера и остановить торговлю.

4. **Интеграция кошельков**
   - финализация `WalletManager` для работы с N кошельками;
   - устранение текущих заглушек и фиксация логики распределения.

## Исключено из MVP

- Flash Volume;
- Jupiter Swaps (выполняются вручную);
- Rent Recovery (скрипт раз в неделю).
